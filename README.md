# RevvedUp

**RevvedUp** is a multiplayer car driving game built with **Unreal Engine**. Players can create and join rooms to compete with others in real-time. Drive freely, explore tracks, and enjoy the thrill of racing with friends or other players online.

---

## Features

* **Multiplayer Rooms**

  * Create your own game room as **Public** or **Private**.
  * Invite friends to private rooms or let anyone join public rooms.

* **Real-Time Gameplay**

  * Once the host starts the game, all players in the room can drive simultaneously.
  * Drive as much as you want—currently, game-ending logic and lap/ring counting are under development.

* **Custom Vehicles**

  * Cars were created with **Blender** and integrated using the **Unreal Engine Vehicle Template**.

* **Network Support**

  * Multiplayer rooms powered by **AdvancedSessions** and **AdvancedSteamSessions**, enabling smooth online gameplay.

* **Immersive Assets**

  * Includes **Fab**, **Megascans**, and **Surfaces** for realistic environments.

---

## Requirements

* **Unreal Engine Version:** **5.6.1** *(required)*

  * The project does **not work on newer versions** like 5.7.3 due to plugin incompatibilities.
* **Plugins:**

  * AdvancedSessions
  * AdvancedSteamSessions

---

## Installation

1. Clone or download the repository.
2. Open the project in **Unreal Engine 5.6.1**.
3. Make sure the required plugins are enabled.
4. Launch the game and navigate to the main menu.
5. Create or join a room to start driving.

---

## Controls

* Use standard vehicle controls (WASD / Arrow Keys) to drive.
* Steering, acceleration, and braking follow Unreal Engine’s Vehicle Template defaults.
